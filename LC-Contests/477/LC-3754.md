## **Concatenate Non-Zero Digits & Multiply by Sum — Summary**

**Task:**
Given an integer **n**, extract all **non-zero digits** *in the same order*, form a new number **x**, compute the **sum of digits of x**, and return **x × sum**.

**Example:**
n = 10203004 → non-zero digits: 1,2,3,4 → x = 1234 → sum = 10 → output = **12340**

---

## **Your Logic (Short Description)**

* Loop through digits from right to left.
* Build the number **x** by adding only non-zero digits.
* Keep track of place value using **i** (1, 10, 100...).
* Also keep a running **sum of all extracted digits**.
* Return `x * sum`.

---

## **Your Code (Cleaned Version)**

```js
var sumAndMultiply = function(n) {
    let sum = 0, num = 0, place = 1;

    while (n) {
        let d = n % 10;
        if (d !== 0) {
            num = d * place + num;
            place *= 10;
            sum += d;
        }
        n = Math.floor(n / 10);
    }
    return num * sum;
};
